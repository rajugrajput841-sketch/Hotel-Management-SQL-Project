# 🏨 Hotel Management System (SQL Project)

## 📌 Project Overview

This project is a **relational database system** designed to manage hotel operations efficiently. It includes customer management, room booking, staff handling, and payment tracking.

The project demonstrates strong understanding of **SQL, database design, and data analysis**.

---

## 🧠 Skills Demonstrated

* SQL (DDL, DML, Joins, Aggregations)
* Database Design & Normalization
* Relational Database Management
* Data Analysis using SQL

---

## 🗂️ Database Structure

The system consists of the following tables:

* **Customers** → Stores customer details
* **Rooms** → Room types, pricing, and capacity
* **Staff** → Staff information
* **Bookings** → Booking transactions
* **Payments** → Payment records

---

## 🔗 Relationships

* One **Customer** → Many **Bookings**
* One **Room** → Many **Bookings**
* One **Staff** → Many **Bookings**
* One **Booking** → One **Payment**

---

## 📊 Key Features

* Customer management system
* Room booking system
* Payment tracking
* SQL-based data analysis

---

## 📈 Sample Analysis Queries

* Total revenue calculation
* Most booked room type
* Top customers by bookings
* Revenue by room category
* Monthly booking trends

---

## 🚀 How to Run the Project
